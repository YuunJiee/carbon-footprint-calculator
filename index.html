名詞解釋

1. 什麼是 TDP (熱設計功耗)？

TDP (Thermal Design Power) 是處理器（CPU/GPU）在最大負載下所產生的熱量，單位為瓦 (W)。它通常被用來近似估算處理器的最大功耗。此處的 TDP 是指單一核心或單一 GPU 的功耗。

2. 什麼是 PSF (務實擴展因子)？

PSF (Pragmatic Scaling Factor) 是一個乘數，用來估算在實際研究中，一個演算法因除錯、參數調整、重複實驗等原因而需要執行的總次數。例如，若一個模型最終需要訓練 1 次，但在過程中您跑了 10 次來找最佳參數，PSF 就會是 10。

3. 如何取得「使用率」？

您可以透過系統的監控工具來取得。在 Windows 中是「工作管理員」的效能分頁；在 macOS 中是「活動監視器」；在 Linux 中則可以使用 `top` 或 `htop` 指令。請在運算任務執行時，觀察 CPU 或 GPU 的平均使用率。

4. 換算單位說明

右側的換算指標（如衛生紙、瓶裝水、啤酒等）皆是指「生產一個單位」所產生的碳排量。
